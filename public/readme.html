Getting started

1. Press the Create button above.

2. Edit the dictionary table.
Note that a dictionary table must have more than two columns, and a title row.

3. Press the Start test button.

4. Then the first question will be displayed. Choose the correct answer between options.

5. If you choose a wrong answer, the correct answer will be displayed, and this question will be repeated again later.

6. You can change the column of question.

7. And question and answer will be reversed.
